Sistema Capítulo Inspetorial

O sistema do Capítulo Inspetorial foi desenvolvido com o objetivo de ser uma plataforma leve e versátil, com várias funcionalidades, a principal proposta do projeto é organizar todos os conteúdos do evento (imagens, PDF/DOCx) " Capítulo Inspetorial " em uma única plataforma segura e restrita com acesso apenas aos participantes do evento. Utilizando tecnologias como PHP, JavaScript e Bootstrap, este projeto aborda a gestão de usuários por meio de um seguro processo de login, área administrativa por onde os usuários administradores podem inserir conteúdo e gerenciar usuários. o evento aconteceu do dia 09 ao dia 12 de Abril, mas a plataforma continua ativa para os usuários que participaram do evento, e será utilizada em eventos similares.

Funcionalidades

  • Área de Login e senha.
  • Dashboard do Usuário com acesso ao conteúdo disponibilizado.
  • Dahsboard para Administradores, onde o conteúdo é iserido e feita gestão de usuários.
